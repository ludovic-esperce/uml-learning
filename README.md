# Diagrammes UML

Ce dépôt contient un ensemble d'exercices d'entraînement à la création de diagrammes UML.

Il couvre les diagrammes UML suivants :
- diagrammes statiques :
  - cas d'utilisation
  - classes
  - déploiement
- diagrammes dynamiques :
  - activité
  - séquence

## Outil de conception

Pour mener à la construction de vos diagrammes vous pourrez utiliser l'outil de votre choix, voici une liste non exhaustive :
- éditeurs [WYSIWYG ("What you see is what you get")](https://fr.wikipedia.org/wiki/What_you_see_is_what_you_get) :
  - [Umlet](https://www.umlet.com)
  - [BOUML](https://www.bouml.fr/)
- édition de diagrammes à partir de langage textuel :
  - [PlantUML](https://plantuml.com/fr)
  - [Mermaid](https://mermaid.js.org/)

> [!TIP]
> Si vous débutez l'UML il vous est conseillé d'utiliser un éditeur WYSIWYG.
> Umlet, de part sa facilité d'utilisation est un bon choix.
