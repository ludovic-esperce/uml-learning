# Diagrammes UML

Ce dépôt contient un ensemble d'exercices d'entraînement à la ccréation de diagrammes UML.

Il couvre les diagrammes UML suivants :
- diagrammes statiques :
  - cas d'utilisation
  - classes
  - déploiement
- diagrammes dynamiques :
  - activité
  - séquence

## Outil de conception

Pour mener à la construction de vos diagrammes vous pourrez utiliser l'outil de votre choix, voici une liste non exhaustive :
- [Umlet](https://www.umlet.com)
